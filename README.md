# 「Who`s Your Wizard」
これは部内短期ハッカソンのプロダクトです。
テーマは「ハリーポッターAPI」です。

# 概要
このプロダクトは、モバイルで写真を撮り、その顔がハリーポッターのキャラの誰に一番似ているかを表示するものです。
使用したフレームワークはVisionです。

#結果
このプロダクトで、部内で最優秀賞を取り、ハリーポッター賞を受賞しました。
